# Hardware NPI Ideation Agent Spec

## Problem

Help hardware product, engineering, manufacturing, and operations stakeholders convert a rough product idea into a structured early NPI planning package.

## Capabilities

- gather and categorize early hardware requirements
- generate stakeholder clarification questions
- define ambiguous requirement language
- identify NPI execution risks
- create a high-level NPI timeline
- export a Markdown summary for review

## External APIs and Data Sources

V1 uses Gemini through the Gemini API when `GEMINI_API_KEY` is configured. The app defaults to `gemini-2.5-flash`, a Flash model intended for fast, cost-conscious use. The workflow also includes deterministic fallback behavior so the public demo remains usable without a key.

The NPI knowledge layer is local and static for v1.

## Safety Constraints

- warn users not to enter confidential hardware, customer, supplier, cost, or credential data
- redact API-key-like and secret-like strings
- avoid committing secrets
- provide export review messaging

## Deployment Preference

Prototype-first public demo. V1 targets Gradio and is suitable for deployment on a public Hugging Face Space or similar free web demo host.
